# 🫐 berry bot

- a [discord app](https://discord.com/developers/docs/quick-start/overview-of-apps) made for [Gentle Berry's Server](https://discord.gg/2Cge3Z7)
- it....looks pretty, like a nice little shelf decoration that does nothing!


## 🏡 what is "berry bot 🫐" monorepo?

- `berry-bot-discord`: the main discord app
   - **berry bot 🫐** (code name "`berry-bot`") is a discord app made for [Gentle Berry's Server](https://discord.gg/2Cge3Z7)
      - here are the following tools that are used:
         - [pycord](https://pycord.dev/)
         - ..[and the lovely dependencies](./berry-bot-discord/requirements.txt)
- *for more information, please see [flufflestack.nuzzles.dev/projects/berry-bot](https://flufflestack.nuzzles.dev/projects/berry-bot)*
